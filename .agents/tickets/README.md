# Frozen archive — pre-GitHub Issue drafts

These files are **planning history, not live work.** They were drafted before this repo was
on GitHub, when an Issue was a local file (the "Ticket" crutch — see ADR-0008).

At the **GitHub cutover** (ADR-0011) the unit of work became a real **GitHub Issue**
produced by `to-issues`. These drafts were **not migrated** to GitHub: they describe
already-shipped or withdrawn Phase 1 work (`01`, `03`, `04` completed; `02` withdrawn), so
re-creating them as Issues would only add closed-on-arrival noise.

- **New work lives in GitHub Issues**, not here. Do not add files to this directory.
- These files are referenced by historical ADRs and reports; they stay in place, read-only.
- The glossary term is **Issue** everywhere now; "Ticket" survives only in this archive and
  the historical ADRs/reports that predate the sweep.
